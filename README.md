# BERTopic-Code-Demo

This repository contains the code used to perform topic modeling on open-ended survey responses from the 2024 SERU (Student Experience in the Research University) dataset using BERTopic.

## Files Included

*model-setup.py*: This file contains the code for initial model set up, the grid search for optimal parameters, and evaluation of the models through a composite score.

*analyze-best.py*: This file contains the code for analyzing the results of the model with optimal parameters, including using OpenAI for representation and visualizing the identified topics.
